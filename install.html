Install on iOS



1. Find the share icon, like you want to share the page
2. Then click on Add to home screen
3. Name the app and click done
4. Bravo! You have the app on your phone, go ahead and create an account.



Install on Android



1. Find the three dot (menu icon)
2. Then click on Install app and confirm
3. The app would start and complete download in a few seconds
4. Bravo! You have the app on your phone, go ahead and create an account.
